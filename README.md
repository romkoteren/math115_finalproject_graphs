# math115_finalproject_graphs
Дискретна математика. Алгоритм Уоршелла (побудова матриці досяжності). Генерація випадкових графів та експериментальна частина.
Виконав Терентьєв Роман 5 група

На репозиторії знаходиться папка з файлами: 
experiment.py (код, який був використаний для експериментів та одержані результати у форматі .csv та .png

interface.py Інтерфейс з генерацією графів, який можна запустити 

Функціонал
1. Генерація графів: Випадкові графи (модель Ердеша-Реньї) із заданою кількістю вершин та щільністю.
2. Алгоритм Уоршелла: Побудова матриці досяжності.
3. Візуалізація: Інтерактивний графік з можливістю переміщення вершин.
